# Task Manager Web App (Full Stack)

This is a full stack web application that allows users to manage daily tasks.

## Features
- Add tasks
- View tasks
- Delete tasks
- REST API integration

## Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

## Description
This project is a full stack task management application where the frontend interacts with a backend server through REST APIs. The backend handles requests and manages task data.

## API Endpoints
- GET /tasks → Fetch all tasks
- POST /tasks → Add new task
- DELETE /tasks/:id → Delete task
